DIVER Export Readme

This compressed file export contains the selected data table, as well as various metadata and supporting information files. The potential file types are described below.

***Main Folder***
Exports of key data from each Data Category (.csv)
Samples and Bioassay data will include references (.csv), qualifier codes (.csv), and study notes (.txt) associated with the exported data.  Bioassay will contain data from individual test replicates (.csv) where available.  Any spatial filter used will be exported as a shapefile (.shp).

***Metadata Folder***
ISO19115 metadata files are included for all DIVER exports. These include DIVER contact information, geographic specifications, and field definitions.

For each Export in main folder
- XML in ISO19115 standard
- XML feature catalogue in ISO19115 standard
- Human-readable HTML in ISO19115 standard

***Related_Data Folder***
Depending on the export, additional related data files may be included. For field observations, this may include electronic transcriptions of field forms associated with the collection activity. Related data from other data categories (e.g., chemistry data for Samples associated with requested Bioassay data) are also in this folder.
